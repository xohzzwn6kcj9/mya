# MYA 테마 시스템 디자인

## 개요

12개 테마를 날짜 기반 정규분포 확률로 선택하는 시스템

## 테마 목록

### 기본 테마 (4종)
1. **네온 사이버펑크** - 어두운 배경 + 형광 네온
2. **선셋 그라데이션** - 노을 배경 + 따뜻한 톤
3. **홀로그램/오로라** - 무지개빛 그라데이션
4. **파스텔 팝** - 연한 파스텔 배경 + 비비드 글자

### 8계절 테마
| 계절 | 기간 | 컨셉 |
|------|------|------|
| 겨울 | 1/1 ~ 2/14 | 스노우, 아이스 블루 |
| 초봄 | 2/15 ~ 3/31 | 라벤더 + 민트 |
| 봄 | 4/1 ~ 5/15 | 벚꽃 핑크 + 새싹 |
| 초여름 | 5/16 ~ 6/30 | 청록 + 라임 |
| 여름 | 7/1 ~ 8/14 | 바다, 시안 |
| 초가을 | 8/15 ~ 9/30 | 골드 + 청록 |
| 가을 | 10/1 ~ 11/14 | 단풍 오렌지 |
| 초겨울 | 11/15 ~ 12/31 | 베이지 + 그레이블루 |

## 확률 분배

### 전체 분배
- 8계절 테마: **80%**
- 파스텔 팝: **10%**
- 네온/선셋/홀로그램: **10%** (각 3.33%)

### 8계절 내부 정규분포 (80% 내에서)
| 거리 | 계절 수 | 각 확률 | 소계 |
|------|---------|---------|------|
| 0 (현재) | 1개 | 60% | 60% |
| 1 (인접) | 2개 | 14% | 28% |
| 2 | 2개 | 4% | 8% |
| 3 | 2개 | 1.5% | 3% |
| 4 (정반대) | 1개 | 1% | 1% |

## 테마 변경 트리거

1. **새로고침**: 항상 테마 재선택
2. **클릭**: 10% 확률로 테마 변경

## 구현 파일

- `src/lib/config/themes.ts` - 테마 정의
- `src/lib/utils/themeUtils.ts` - 테마 선택 로직
- `src/lib/config/displayOptions.ts` - 기존 파일 수정
